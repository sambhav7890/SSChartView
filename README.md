## Installation

SSChartView is not available through [CocoaPods](http://cocoapods.org). 

To install it:

1. Add the following line to your Podfile:

```ruby
pod "SSChartView"
```

2. Add the podspec to your pod specs repo
```
pod repo push <YourRepoName> SSChartView.podspec
```
